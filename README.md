# 📊 Data Jobs Market Analysis Project

## 📌 Overview
This project analyzes a dataset of data-related job roles to understand trends in job demand, required skills, and experience levels in the data industry.

The goal is to identify which roles and skills are most in demand for entry-level candidates.

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📊 Key Insights
- Data Analyst roles are the most frequently occurring in the dataset.
- SQL and Python are the most in-demand skills across all roles.
- Entry-level positions still require strong technical foundations.
- Data Engineering roles often require cloud and programming skills.

---

## 📁 Project Structure
- Data creation using Python dictionaries
- Data analysis using Pandas
- Data visualization using Matplotlib
- Insight generation from data
